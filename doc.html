Load script

		$loader->load('countries')->init(<ms>,<version>,<url>);

ms - values are global, businesses, cars, deals, events, jobs, realestate. Default to global

version - version of the db, if changed, db will reset and pull the countries data back to db again

Then you need to set up your templates for countries, states, cities, inside its corresponding wrappers, below is a sample

		<div class="country-wrapper">
			<div class="country" data-template="#country-template">
				<div class="template">
					<select name="country">
						<option value="" selected>Select country</option>
						{options}
					</select>
				</div>
			</div>
			<div class="state" data-template="#state-template">
				<div class="template">
					<select name="state">
						<option value="" selected>Select state</option>
						{options}
					</select>
				</div>
			</div>
			<div class="city" data-template="#city-template">
				<div class="template">
					<select name="city">
						<option value="" selected>Select city</option>
						{options}
					</select>
				</div>
			</div>
		</div>

{options} - this where all options will be put in

For country options

	<div class="country" data-template="#country-template"></div>

For state options

	<div class="state" data-template="#state-template"></div>

For city options

	<div class="city" data-template="#city-template"></div>

You can add your pre-selected item by adding data-default, either text or id e.g

	<div class="country" data-template="#country-template" data-default="Nigeria"></div>

You can also hide others country and just show your default by adding hide-others class to country component e.g.

	<div class="country hide-others" data-default="Nigeria" data-template="#country-template"></div>
